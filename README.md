# InsightFlow - Automated Data Analytics Dashboard

![Dashboard Preview](static/img/preview.png)

InsightFlow is a premium, automated data analysis and visualization system designed to turn raw CSV and Excel data into interactive, actionable insights. Built with a modern **Neumorphic UI**, it focuses on a premium user experience and effortless data exploration.

## 🚀 Key Features

- **Automated Analysis**: Upload any CSV or Excel file and let the engine automatically calculate summaries, distributions, and trends.
- **Interactive Visualizations**: High-quality charts (Bar, Line, Pie, Scatter, etc.) that can be toggled and explored in real-time.
- **Statistical Summaries**: Instant overview of row counts, missing values, and key column statistics.
- **Professional PDF Reports**: Export your findings into a clean, well-formatted PDF report with just one click.
- **Premium Neumorphic Design**: A sleek, dark-mode interface designed for a modern aesthetic.
- **Fully Responsive**: Optimized for mobile, tablet, and desktop viewports.

## 🛠️ Tech Stack

- **Backend**: Python (Flask)
- **Data Processing**: Pandas, NumPy
- **Visualization**: Chart.js
- **PDF Generation**: FPDF2
- **Frontend**: HTML5, Vanilla CSS3, Javascript (ES6+)

## 📦 Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Rudra9069/InsightFlow-Data-Analytics-.git
   cd InsightFlow-Data-Analytics-
   ```

2. **Set up a virtual environment (optional but recommended):**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use: venv\Scripts\activate
   ```

3. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the application:**
   ```bash
   python app.py
   ```
   The dashboard will be available at `http://localhost:5000`.
